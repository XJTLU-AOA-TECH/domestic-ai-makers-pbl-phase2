Blender + AI 指导手册

基于 dcc-mcp-blender 的完整安装与使用指南

📦 前置要求

  • Blender 4.2+ 或 5.0（推荐最新版）
  • Claude Code（命令行版）
  • Python 3.10+（Blender 自带）

🔧 安装步骤

Step 1：下载插件

前往 GitHub Releases 下载对应平台的 ZIP 文件：

https://github.com/dcc-mcp/dcc-mcp-blender/releases

下载对应你操作系统的文件：

操作系统文件名
Windowsdcc_mcp_blender_addon_win64_vX.Y.Z.zip
macOSdcc_mcp_blender_addon_macos_vX.Y.Z.zip
Linuxdcc_mcp_blender_addon_linux_vX.Y.Z.zip

Step 2：安装插件（Blender 4.2+）

1

打开 Blender → Edit → Preferences

确保 Blender 版本 ≥ 4.2

2

切换到 Extensions 标签页

点击 Install from Disk…（从磁盘安装）

3

选择下载的 ZIP 文件

Blender 会自动解压并安装依赖到隔离环境

4

勾选启用 DCC MCP Blender

安装完成后，在列表中找到并勾选启用

5

确认服务启动

MCP Server 会自动在 http://127.0.0.1:8765 启动

注意：这是 Blender 4.2+ 的 Extension 格式，包含 blender_manifest.toml。不要按老方法走 Edit → Preferences → Add-ons → Install，那会报错！

Step 3：配置 Claude Code

在 Claude Code 对话中直接输入以下命令：

# 添加 Blender MCP Server（HTTP 传输模式） claude mcp add --transport http blender http://127.0.0.1:8765/mcp # 如果提示已存在，先删除再添加 claude mcp remove blender claude mcp add --transport http blender http://127.0.0.1:8765/mcp # 验证是否添加成功 claude mcp list
添加成功后，在 Claude Code 对话中输入 你有什么工具？，确认 Blender 出现且状态正常。

Step 4：验证连接

确保 Blender 已打开且插件已启用，然后在 Claude Code 中输入：

列出当前场景中所有物体的名称和类型

如果 Claude 能正确返回场景信息，说明 MCP 连接成功！🎉

❓ 常见问题

问题解决方案
安装 ZIP 时报错确认下载的是 Release 里的平台 ZIP，不是源码 ZIP。去 Releases 页面下载 dcc_mcp_blender_addon_xxx_vX.Y.Z.zip
Claude 找不到 Blender 工具1. 检查 Blender 中插件是否已启用
2. 确认 Blender 窗口处于焦点状态
3. 重启 Claude Code
4. 检查 claude mcp list 是否显示 blender
连接超时检查防火墙是否阻止 8765 端口；确保 Blender 窗口处于焦点状态
AI 生成的模型面数太多用 Blender 的 Decimate 修改器或 Remesh 减少面数，然后手动重拓扑
导出 STL 后打印失败使用 Blender 的 3D Print Toolbox 插件检查壁厚、非流形边、悬空面
执行脚本时一直弹授权框在 Claude Code 中输入：/claude config set autoApproveTools blender
Windows 下 uvx 报错改用 cmd + /c 方式调用，或直接使用 HTTP URL 配置
