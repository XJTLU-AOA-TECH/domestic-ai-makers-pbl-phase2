## 切片是什么？为什么需要它？

3D 打印机听不懂 STEP 或 STL。它只懂一种语言：**G-code**——告诉喷头"走到哪、挤多少料、温度多少"。

**切片（Slicing）** = 把 3D 模型翻译成 G-code 的过程。就像编译器把高级代码翻译成机器码。

```
3D 模型（STL） → 切片软件 → G-code → 打印机执行
     ↑              ↑           ↑
  设计师        你在这里    机器语言
```

切片软件要决定：
- **层厚**：0.1mm 精细但慢，0.3mm 快速但粗糙
- **填充密度**：100% 实心 vs 20% 蜂窝（省料）
- **支撑**：悬空部分要不要加支架？
- **壁厚**：外壳打几层？
- **温度**：喷头 200°C，热床 60°C

> **核心原则**：切片不是"一键导出"，而是制造约束前置——你在设计时就要考虑壁厚、悬垂、支撑，否则切片后要么打不了，要么废品。

---

## Bambu Studio 是什么？

Bambu Studio 是 Bambu Lab 推出的免费切片软件，基于开源的 PrusaSlicer，但针对 Bambu 打印机做了深度优化。

| 功能 | 说明 |
|-----|------|
| 导入 STL/STEP/3MF | 直接加载模型 |
| 自动切片 | 一键生成 G-code |
| 支撑生成 | 自动检测悬空并加支撑 |
| 打印预览 | 逐层模拟打印过程 |
| 局域网发送 | 直接推送到打印机 |

> **注意**：Bambu Studio 也可以给其他品牌打印机切片（如 Prusa、Ender），只是预设参数不同。

## 实战：给上午的 5 个零件做切片

### 步骤 1：准备 STL 文件

上午生成的 STEP 文件需要先转成 STL（三角网格，打印机认识的格式）：

```bash
# 零件 1：校准块
npx skills run cad --input ./benchmark_01_block.step --export stl --output ./benchmark_01_block.stl

# 零件 2：法兰盘
npx skills run cad --input ./benchmark_02_flange.step --export stl --output ./benchmark_02_flange.stl

# 零件 3：L 支架
npx skills run cad --input ./benchmark_03_lbracket.step --export stl --output ./benchmark_03_lbracket.stl

# 零件 4：阶梯轴
npx skills run cad --input ./benchmark_04_shaft.step --export stl --output ./benchmark_04_shaft.stl

# 零件 5：电子外壳
npx skills run cad --input ./benchmark_05_enclosure.step --export stl --output ./benchmark_05_enclosure.stl
```

> **科普**：STEP 转 STL 是"降维"——从参数化模型变成三角网格。这个过程中圆会变成多边形、曲面会分段，所以 STL 的分辨率（三角面数量）要适中：太少会看出棱角，太多文件巨大。

---

### 步骤 2：导入 Bambu Studio

1. 打开 Bambu Studio
2. 点击 **"新建项目"** 或 **Ctrl+N**
3. 点击 **"添加"** → **"加载模型"**
4. 选择 `benchmark_01_block.stl`

---

### 步骤 3：摆放与检查

**摆放（Orientation）**：
- 模型导入后可能方向不对。选中模型 → 点击工具栏 **旋转** 图标 → 调整角度
- **原则**：最大的平面朝下（接触热床），减少支撑

**检查（Manifold）**：
- 点击右侧 **"修复"** 按钮，检查模型是否有破面、非流形边
- 如果 AI 生成的模型有缺陷，这里会标红

> **思考**：上午 AI 生成的零件，导入后有没有报错？如果有，说明生成质量还不够完美——这也是 Loop Engineering 需要检查的一环。

---

### 步骤 4：设置切片参数

右侧参数面板，重点调这几个：

| 参数 | 推荐值 | 说明 |
|-----|-------|------|
| **层厚** | 0.2mm | 平衡速度与精度 |
| **壁厚** | 3 层 | 外壳强度 |
| **填充密度** | 20% | 内部蜂窝，省料 |
| **填充图案** | 网格/陀螺 | 强度与速度平衡 |
| **支撑** | 自动 | 检测悬空 > 45° 自动加支撑 |
| **喷头温度** | 200°C | PLA 标准 |
| **热床温度** | 60°C | PLA 标准 |

**逐零件分析**：

| 零件 | 特殊注意 | 为什么 |
|-----|---------|-------|
| **校准块** | 无需支撑，平放即可 | 结构简单，无悬空 |
| **法兰盘** | 中心孔朝下，减少支撑 | 圆柱孔如果朝上需要内部支撑 |
| **L 支架** | 加强筋方向决定摆放 | 避免支撑挡住筋板 |
| **阶梯轴** | 竖直打印（Z 轴） | 水平打印层间强度差，易断 |
| **电子外壳** | 开口朝上，壁厚检查 | 薄壁（3mm）容易打穿或变形 |

---

### 步骤 5：切片预览

点击工具栏 **"切片"** 按钮（或 **Ctrl+G**）。

Bambu Studio 会：
1. 计算每一层的喷头路径
2. 显示预计打印时间、耗材用量
3. 打开 **预览模式**，你可以逐层拖动滑块看打印过程

**重点观察**：
- 悬空部分有没有生成支撑？（紫色线条）
- 壁厚是否均匀？
- 细小特征（如孔、筋）是否被切片算法"吞掉"了？

> **如果发现孔被填满了**——说明 STL 的孔有问题，或者孔径太小（< 2mm 打印机打不出来）。回到上午的模型检查 Prompt 是否精确。

---

### 步骤 6：导出 G-code

点击 **"导出 G-code"** → 保存为 `.gcode` 文件。

```
benchmark_01_block.gcode
benchmark_02_flange.gcode
...
```

> **G-code 科普**：打开 `.gcode` 文件，你会看到一行行指令：
> ```
> G1 X50 Y30 E1.5 F1500   ; 移动到 (50,30)，挤出 1.5mm 料，速度 1500
> M104 S200                ; 设置喷头温度 200°C
> ```
> 这就是 3D 打印机的"机器语言"。

## 进阶：制造约束自查表

切片不是终点，而是制造约束的"终审"。用这个表检查每个零件：

| 检查项 | 合格标准 | 你的零件 |
|-------|---------|---------|
| 最小壁厚 | ≥ 1.2mm（ nozzle 0.4mm × 3 层） | □ |
| 悬空角度 | ≤ 45° 无需支撑，> 45° 需支撑 | □ |
| 孔径下限 | ≥ 2mm（太小打不圆） | □ |
| 层高匹配 | 特征高度是层厚的整数倍 | □ |
| 支撑可拆 | 支撑不能封死内部空腔 | □ |
| 打印方向 | 最大平面朝下，强度方向沿 Z | □ |

**如果某一项不合格**——说明上午的 Prompt 需要修正，或者 Loop Engineering 的验收标准要加入"制造约束检查"。

## 故障速查

| 问题 | 原因 | 解决 |
|-----|------|------|
| 模型导入报错 | STL 有破面/非流形 | 用 Bambu Studio 修复，或回炉重生成 |
| 切片后孔被填满 | 孔径太小或 STL 缺陷 | 检查 Prompt 孔径 ≥ 2mm，重新导出 STL |
| 悬空部分没支撑 | 支撑阈值设太高 | 把支撑角度阈值从 45° 调到 50° |
| 预计时间太长 | 填充密度太高或层厚太薄 | 填充改 15%，层厚改 0.28mm |
| 薄壁打不圆 | 壁厚不是 nozzle 的整数倍 | 壁厚设为 0.4/0.8/1.2/1.6mm 等 |
