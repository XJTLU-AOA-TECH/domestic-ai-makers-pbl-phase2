什么是 Text2CAD？

💬
自然语言输入
用户用日常语言描述设计需求，如"创建一个带中心孔的矩形块"，无需掌握专业 CAD 软件操作
🧠
AI 理解语义
大语言模型解析文本中的几何特征、尺寸关系、拓扑结构，转化为机器可理解的参数化描述
📐
参数化 CAD 输出
生成可直接编辑的 3D 模型文件（STEP、STL、OBJ 等），保留设计历史与参数，支持后续修改

🎯 核心价值

Text2CAD 打破了设计门槛——让非专业工程师也能快速创建精确的三维模型；提升了设计效率——将概念到原型的时间从数小时缩短至数秒；实现了智能迭代——通过修改文本描述即可调整设计，无需重新建模。

03

技术发展历程

2021
DeepCAD 数据集发布
DeepCAD 数据集诞生，包含约 17 万个参数化 CAD 模型，为后续 Text2CAD 研究奠定了数据基础
2022
扩散模型崛起
Stable Diffusion 等扩散模型在图像生成领域取得突破，研究者开始探索将文本生成图像技术应用于 CAD 领域
2023
多视角生成技术
Zero-1-to-3 等模型实现从单张图像生成多视角视图，为从文本到正交工程图的转换提供关键技术支撑
2024
Text2CAD 框架发布
Sadil Khan 团队在 NeurIPS 2024 (Spotlight) 发布 Text2CAD，首个端到端文本生成参数化 CAD 模型的 AI 框架
2025
工程化落地
earthtojake 推出 CAD Skills 工具库，集成到 Codex 和 Claude Code 等 AI 编程助手，实现从设计到制造的全链路自动化
04

核心技术原理

🔤
文本编码
使用 CLIP 或专用文本编码器将自然语言描述转化为高维语义向量，捕捉几何特征与空间关系
🔄
Transformer 架构
自回归 Transformer 网络逐步生成 CAD 构建序列（草图→拉伸→倒角），像搭积木一样构建 3D 模型
🎨
扩散模型
通过去噪扩散过程生成等轴测视图，再转换为正交工程图，确保几何一致性与工程标准合规
📊
参数化表示
使用 CAD 向量序列（圆、线、样条、拉伸操作等）表示模型，保留可编辑的设计历史与参数
🎯
多任务训练
联合优化视觉质量、参数精度与几何准确性，使用 Chamfer Distance、F1-Score 等指标评估
🌐
多模态融合
结合 LLM（Mistral）和 VLM（LLaVA-NeXT）进行数据标注，实现从图像到文本再到 CAD 的跨模态生成
05

Sadil Khan Text2CAD

📌 项目概况

NeurIPS 2024 Spotlight 论文
作者：Mohammad Sadil Khan 等（DFKI, RPTU, MindGarage）
GitHub: github.com/SadilKhan/Text2CAD
数据集：~170K 模型，~660K 文本标注

🎯 核心创新

1. 首个端到端文本生成参数化 CAD 框架
2. 支持从初学者到专家级的多级文本提示
3. 基于 Transformer 的自回归序列生成
4. 开源数据标注管道（Mistral + LLaVA-NeXT）

⚙️ 技术架构

输入：自然语言文本提示
编码：文本 → 语义向量（CLIP/LLM）
生成：Transformer 自回归生成 CAD 向量序列
输出：参数化 3D CAD 模型（可编辑）
评估：视觉质量 + 参数精度 + 几何准确性

📊 性能指标

在 DeepCAD 测试集上验证，支持多种复杂度的几何体生成，包括拉伸、旋转、倒角、孔特征等常见 CAD 操作。

earthtojake CAD Skills

🚀 项目定位

这是一个面向 AI Agent 的 CAD 技能库，集成到 Codex 和 Claude Code 等编程助手中，实现从自然语言到制造的全链路自动化。不仅仅是生成模型，更涵盖了设计、检查、采购、切片、打印的完整工作流。

📐

CAD 生成

从文本/图像创建 STEP 模型，支持 STL、3MF、GLB 导出

🔍

CAD 查看器

本地浏览器预览 CAD、G-code 和机器人文件

🔩

标准件库

查找螺丝、轴承、电机等现成 STEP 零件

📄

DXF 工程图

生成 2D 轮廓、模板、垫片和切割布局

🤖

URDF 机器人

生成机器人结构文件，包含关节、约束和网格

🖨️

G-code 切片

将网格文件切片为 FDM 打印机可用的 G-code

# Sadil Khan Text2CAD 使用示例
from text2cad import Text2CADGenerator

# 初始化模型
generator = Text2CADGenerator(checkpoint="text2cad-v1.pt")

# 输入文本提示
prompt = "A rectangular prism with a cylindrical hole in the center"

# 生成 CAD 模型
cad_model = generator.generate(prompt)

# 导出为 STEP 文件
cad_model.export("output.step")

未来展望

🧠 多模态融合
结合文本、草图、语音甚至手势输入，让设计意图的表达更加自然直观。AI 将同时理解语言描述与手绘草图，生成更精准的 CAD 模型。
⚡ 实时协作
支持多人同时通过自然语言编辑同一个 CAD 模型，AI 实时协调冲突、建议优化方案，实现"对话式设计"的协作模式。
🏭 制造一体化
Text2CAD 将直接对接 CAM 系统，生成的模型自动进行可制造性分析（DFM）、成本估算，并输出机床可直接执行的加工代码。
🎓 教育普及
降低 CAD 学习门槛，让中学生、设计师、产品经理无需掌握复杂软件即可表达三维创意，推动工程教育的普惠化。
